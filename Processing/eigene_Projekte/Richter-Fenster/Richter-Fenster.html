Das Richter-Fenster

Das Richter-Fenster befindet sich im Südquerhaus des Kölner Dom und deckt eine Fensterfläche von 106m². Auf den ersten Blick scheint es einfach, als seien die verschieden-farbigen Fensterplättchen volkommen zufällig angeordnet, bei näherem Hinsehen fällt jedoch auf, dass eine gewisse Symmetrie vorliegt.
Die erste und die dritte "Spalte", sowie die Zweite und die Fünfte und die Vierte und Sechste, lassen sich vertikal an einer Geraden spiegeln. Gerhard Richter, Konstrukteur und Namensgeber der Fensterstruktur, wird sich also höchst wahrscheinlich ein Programm/Script dieser Art (siehe unten) erstellt haben, um diese zu erschaffen.
Ich habe dies nun in Java Processing rekonstruiert. Mit Hilfe jeweils zweier verschachtelter For-Schleifen werden die gleichfarbigen "Kacheln" an einer y-Achse gespiegelt und bilden so jenes interessante Muster, welches wie erwähnt auch im Kölner Dom zu finden ist.
Durch drüken der Leertaste, kann eine neue zufällige Anordnung der einzelnen Fenster generiert werden.
Richter-Fenster Code
int e;
void setup() {
  size(395, 853);
  noStroke();
  colorMode(HSB, 360, 100, 100);
  Fenster();
}
void draw() {
  if (keyPressed) {
    Fenster();
  }
}
void Fenster() {
  //ganz links (13er Reihe)
  for (int i = 0; i <= 60; i = i + 5) {
    for (int e = 0; e <= 1000; e = e + 5) {
      fill(random(360), random(40, 90), random(40, 90));
      rect(i, e, 5, 5);
      rect(180-i, e, 5, 5);
    }
  }
  //links+1 (11er Reihe)
  for (int i = 65; i <= 115; i = i + 5) {
    for (int e = 0; e <= 1000; e = e + 5) {
      fill(random(360), random(40, 90), random(40, 90));
      rect(i, e, 5, 5);
      rect(390-i, e, 5, 5);
    }
  }
  //Mitte (5er Reihe [ungespiegelt])
  for (int i = 185; i <= 205; i = i + 5) {
    for (int e = 0; e <= 1000; e = e + 5) {
      fill(random(360), random(40, 90), random(40, 90));
      rect(i, e, 5, 5);
    }
  }
  //mitte rechts (13er Reihe)
  for (int i = 210; i <= 270; i = i + 5) {
    for (int e = 0; e <= 1000; e = e + 5) {
      fill(random(360), random(40, 90), random(40, 90));
      rect(i, e, 5, 5);
      rect(600-i, e, 5, 5);
    }
  }
  fill(0);
  rect(55, 0, 10, 1000);
  rect(120, 0, 10, 1000);
  rect(185, 0, 25, 1000);
  rect(265, 0, 10, 1000);
  rect(330, 0, 10, 1000);
}
