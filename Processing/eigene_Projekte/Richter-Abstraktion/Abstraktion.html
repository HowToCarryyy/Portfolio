192 Farben

Das Bild 192 Farben bildet das Cover eines Plakates für die Austellung: ”Abstraktion” von Gerhard Richter aus Mitte 2018. Im Unterschied zu meinem ”Imitat”, befinden sich auf dem Bild 192 verschiedene Farben in den Vierecken.
Da ich bei dem ”Imitat” für die Farben eine Zufallsfunktion verwendet habe, ist es möglich, (dabei jedoch sehr unwahrscheinlich) dass sich gleichfarbige Vierecke bilden.
Mit einem Tastendruck lassen sich die Vierecke neu einfärben. 
192 Farben Code
void setup() {
  size(320, 420);
  noStroke();
  background(220);
  colorMode(HSB, 360, 100, 100);
  Vierecke();
}
void draw() {
  if (keyPressed) {
    Vierecke();
  }
}
void Vierecke() {
  for (int i = 10; i <= 300; i = i + 25) {
    for (int e = 10; e <= 400; e = e + 25) {
      fill(random(360), random(40, 50), random(40, 100));
      rect(i, e, 20, 20);
    }
  }
}
